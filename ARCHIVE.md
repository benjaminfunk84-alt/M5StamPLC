# Archiv: M5StamPLC-Projekt

**Status:** Archiviert (Stand: Februar 2025)

**Grund:** M5StamPLC-Hardware hat sich als defekt erwiesen. Das Projekt wird nicht weiter auf dem StamPLC betrieben.

**Nachfolge:** Die Vorgaben wurden auf **M5CoreS3** mit Zusatzmodulen umgestellt. Siehe **[VORGABEN_M5CoreS3.md](VORGABEN_M5CoreS3.md)** für die neue Architektur, Hardwareliste und Kommunikationsoptionen (WiFi, ESP-NOW, Bluetooth) zum M5Tab5.

Der bestehende Code und die PlatformIO-Konfiguration in diesem Repository bleiben als Referenz erhalten (StamPLC-spezifische Pins, M5StamPLC-API, RS485-Protokoll-Idee). Für das neue Setup ist ein eigenes Projekt/Code auf Basis der neuen Vorgaben vorgesehen.
